Introduction¶

Welcome to CySparse‘s users manual!

We tried to keep this manual as simple and short as possible. You can find much more detailed information in the developer’s manual. To see CySparse in action, you can try the tutorials written in IPython Notebooks or run the examples in the examples directory.

What is CySparse?¶

CySparse is a fast sparse matrix library for Python/Cython.

PySparse legacy¶

CySparse can be seen as the replacement of the PySparse library with lots of improvements. Instead of improving the existing PySparse library written in C and CPython, we decided to rewrite it from scratch in Cython. This allows us a shorter development cycle and to hook existing C and C++ libraries very easily to CySparse (see the PythonOptimizers project for a full list of compatible Solvers wrappers).

In short, CySparse is PySparse but on steroids!

PySparse vs CySparse¶

Even if CySparse is (strongly) inspired from PySparse, there are notable differences. In short, CySparse:

  • allows the use of matrices with different types of indices and elements at run time (see ...);
  • is faster than PySparse (see our benchmarks);
  • uses matrix views - a very light proxy object - that represent parts of a matrix without the need to copy any element (see...);
  • uses matrix proxies - an even lighter proxy object - that represent some common transformation of a matrix (like the transposed of a matrix) without the need to copy any element (see ...);
  • has more syntactic sugar, like A * b, b * A, A.T * b etc.
  • has a symmetric storage scheme for all its matrix types.
  • doesn’t use masks.
  • has lots of unit tests.
  • is well integrated with some of the best solvers (SuiteSparse, MUMPS, qr_mumps, ...).
  • is well documented.
  • can be used in Python and Cython.
  • has less dependencies.

Both libraries define similar but also different matrix classes:

Matrix type PySparse CySparse
Linked-List Format ll_mat, ll_mat_sym, PysparseMatrix LLSparseMatrix
Compressed Sparse Row Format csr_mat CSRSparseMatrix
Compressed Sparse Column Format
CSCSparseMatrix
Sparse Skyline Format sss_mat
Compressed Sparse Row and Column Format
CSBSparseMatrix (later)

What is the maturity level of CySparse?¶

If you don’t mix matrices with different types and use vector multiplication, CySparse has been tested quite heavily but as we only started in 2016, CySparse is still in its infancy. If you stick to the basics, it works. The rest is work in progress. We are planning to add more tests soon and to test extensively its integration with several well known solvers.

Warning

Do not use this library for critical tasks!

The benchmarks also shows that the library is fast, much faster than its competitors.
